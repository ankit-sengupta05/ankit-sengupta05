🕶️ Smart Vision Aid
AI-powered assistive smart glasses — real-time spatial awareness for the visually impaired
🕶️

An end-to-end embedded AI system giving visually impaired users real-time 360° spatial awareness. Three ESP32-CAM modules feed live video into a YOLO object detection pipeline. Ultrasonic sensors provide proximity data. The Android companion app converts all detections into instant audio alerts and turn-by-turn navigation guidance. All firmware runs in C++ for microsecond-level response.

📷
Vision System — 3× ESP32-CAM OV2640 modules + HC-SR04 ultrasonic sensors providing full 360° overlapping spatial coverage
⚡
AI Detection — YOLOv5/v8 real-time object detection; identifies obstacles, people & hazards with precise bounding box coords
📱
Android Companion — receives live Wi-Fi detection data, delivers contextual audio alerts & navigation feedback via TTS
🔧
Embedded Core — interrupt-driven C++ on Arduino IDE; sensor fusion pipeline with <50ms end-to-end latency
🏅
Achievement — Selected for VIT Amaravati Internal Expo 2025 as a standout assistive technology innovation
